## Quick Start

``` bash
# Install dependencies
npm install

# Serve on localhost:3000
npm start


## App Info
React Life Cycle Methods are :
	componentWillMount
	componentDidMount
	shouldComponentUpdate
	componentWillUpdate
	componentDidUpdate
	componentWillReceiveProps
	componentWillUnmount
